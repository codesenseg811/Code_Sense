function fibonacci(n) {
  if (n <= 1) return n;
  return fibonacci(n-1) + fibonacci(n-2);
}
Explanation:

This function calculates the Fibonacci number for a position using recursion. If n is 0 or 1, it returns n; otherwise, it returns the sum of the two previous Fibonacci numbers.
